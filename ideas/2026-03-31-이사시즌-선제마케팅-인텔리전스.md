# 이사 시즌 선제 마케팅 인텔리전스

> 출처: [[3-Resources/해커톤/SNOWFLAKE-KR-HACKATHON-2026]] — 아이디어 2

## 핵심 아이디어

아파트 시세 급등·인구 이동 신호를 감지해 인터넷·렌탈 마케팅 타이밍과 문구를 자동 생성.

**문제 정의**
이사 수요는 아파트 거래가 활발해지는 시점보다 1~2개월 앞서 발생함. 시세 + 유동인구 + 계약 데이터를 결합하면 "지금 집중해야 할 지역"과 "보내야 할 메시지"를 자동화할 수 있음.

## 활용 데이터

| 데이터 | 활용 목적 |
|--------|-----------|
| REGION_APT_RICHGO_MARKET_PRICE_M_H | 시세 급등 지역 탐지 (선행 신호) |
| FLOATING_POPULATION_INFO | 유동인구 급증 → 이사 유입 신호 |
| V01 지역별 계약 통계 | 과거 이사 시즌 계약 패턴 |
| V04 유입 채널별 성과 | 지역별 최적 마케팅 채널 파악 |
| REGION_MOIS_POPULATION_GENDER_AGE_M_H | 연령대별 이사 수요 타겟 세분화 |

## Snowflake / AI 활용

- **Cortex Anomaly Detection**: 시세·유동인구 이상 급등 지역 자동 탐지
- **Cortex COMPLETE (LLM)**: 탐지된 지역·고객 프로파일 기반 마케팅 문구 자동 생성
- **Streamlit**: 이사 수요 경보 지도 + LLM 생성 문구 미리보기

## 심사 포인트

- **창의성**: 시세 선행 신호 → 마케팅 자동화로 연결하는 파이프라인
- **AI**: Cortex 2종 활용 (Anomaly Detection + LLM) → 탐지→생성 가치 사슬
- **발표**: "데이터가 마케터의 역할을 대신한다"는 스토리라인 명확
