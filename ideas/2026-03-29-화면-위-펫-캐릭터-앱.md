# 화면 위를 돌아다니는 펫 캐릭터 앱

맥북 앱 또는 크롬 익스텐션으로 강아지/고양이 캐릭터가 화면 위를 자유롭게 돌아다니는 기능.

## 핵심 아이디어

- **플랫폼**: macOS 데스크톱 앱 또는 Chrome Extension
- **컨셉**: 화면 위에 오버레이로 펫 캐릭터가 돌아다님 (일본의 "시메지" 앱 유사)
- **캐릭터**: 강아지, 고양이 등 귀여운 픽셀/일러스트 캐릭터
- **인터랙션**: 클릭하면 반응, 먹이 주기, 기분 상태 등

## 구현 방향

### macOS 앱
- SwiftUI + `NSWindow` (투명 + always-on-top 윈도우)
- 스프라이트 애니메이션으로 이동/행동 표현
- 메뉴바 아이콘으로 설정 접근

### Chrome Extension
- `chrome.windows` + 투명 오버레이 레이어
- Canvas / CSS 스프라이트 애니메이션
- 모든 페이지 위에 content script로 삽입

## 참고 레퍼런스

- **시메지(Shimeji)**: Java 기반 데스크톱 펫 (원조 컨셉)
- **Desktop Goose**: 거위가 화면을 돌아다니는 앱 (macOS/Windows)
- **Tamagotchi 감성**: 돌봄/성장 요소 추가 가능

## 수익화 가능성

- 캐릭터 스킨 유료 판매
- Chrome Web Store 유료 앱
- 구독형 (월정액으로 신규 캐릭터 제공)
