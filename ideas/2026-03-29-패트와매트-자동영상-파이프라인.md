# 패트와 매트 컨셉 자동 영상 파이프라인

친구 2명이서 **패트와 매트** 스타일의 짧은 영상을 자동으로 생성하는 파이프라인 구축 아이디어.

## 핵심 아이디어

- **컨셉**: 패트와 매트처럼 두 캐릭터가 엉뚱하고 창의적인 방식으로 문제를 해결하는 짧은 영상
- **제작 방식**: AI 영상 생성 도구 활용 (Kling AI 또는 Hailuo 등)
- **팀 구성**: 친구 2명이 역할 분담하여 운영
- **목표**: 수동 작업 최소화, 파이프라인 자동화

## 파이프라인 구성 요소 (초안)

1. **스크립트 생성** → Claude/GPT로 에피소드 아이디어 자동 생성
2. **영상 생성** → Kling AI 또는 Hailuo로 씬 단위 영상 생성
3. **편집 자동화** → FFmpeg 또는 CapCut API로 자동 편집/자막
4. **업로드** → YouTube Shorts / TikTok / 인스타 릴스 자동 배포

## 고려 사항

- Kling AI vs Hailuo 비교 필요 (품질, 가격, API 지원 여부)
- 캐릭터 일관성 유지 방법 (동일 캐릭터가 에피소드마다 등장)
- 저작권 문제 없는 독자 캐릭터 디자인 필요
- 2인 워크플로우 분담 구조 설계
