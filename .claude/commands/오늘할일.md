# 오늘 할일

오늘의 작업 명세서를 생성하고 출력합니다.

## 절차

### 1. 본인 확인
`config.local.json`에서 member를 읽어줘.
파일이 없으면 "config.local.json이 없습니다. config.local.json.example을 복사해서 member를 설정해주세요." 안내 후 중단.

### 2. 오늘 명세서 확인 및 자동 생성
오늘 날짜(YYYY-MM-DD)를 구해줘.

`tasks/today-{날짜}-{member}.md` 파일이 이미 있으면 3단계로 바로 이동해줘.

없으면 아래 절차로 자동 생성해줘:

  a. `events.jsonl`에서 해당 member의 **모든** `task_done` 이벤트 task_id를 추출해줘 (날짜 무관).
  b. `projects/` 폴더의 모든 .json 파일을 읽어서 해당 member가 담당하는 태스크만 추려줘.
     아래 조건 중 하나라도 해당하면 제외해줘:
     - 태스크 JSON에 `"status": "done"` 이 있는 경우
     - a에서 추출한 task_done 목록에 해당 task_id가 있는 경우 (JSON 미반영 완료 태스크 방지)
  c. 아래 형식으로 `tasks/today-{날짜}-{member}.md`를 생성해줘:

  ```
  # 오늘 할 일 — {member} / {날짜}

  ## 작업 목록

  ### {순번}. {task_id}: {title} [{priority}]
  - 프로젝트: {project id}
  - 레포: {path}
  - 배경: {background} (없으면 생략)
  - 관련 스킬: {skills} (없으면 생략)
  - 예상 시간: {estimate_h}h
  ```

  d. `events.jsonl`에 오늘 `day_start`가 없는 경우에만 append:
  ```
  {"ts":"{ISO8601 현재시각}","type":"day_start","member":"{member}"}
  ```

### 3. 출력
명세서 전체 내용을 출력해줘.
마지막에 다음 안내를 출력해줘:
"`/작업시작 {순번}` 또는 `/작업시작 {task_id}` 로 시작하세요."
