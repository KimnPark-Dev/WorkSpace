# 전체 풀

WorkSpace 루트와 모든 서브모듈을 최신으로 동기화합니다.

## 절차

### 1. WorkSpace 루트 풀
```bash
git pull
```
실패하면 오류 메시지를 출력하고 계속 진행해줘 (다른 레포에 영향 없음).

### 2. 서브모듈 최신화
```bash
git submodule update --remote --merge
```
각 서브모듈(LLM_wiki, AlgoNotion_FE, AlgoNotion_BE, TalkTime)의 원격 최신 커밋을 가져와 머지해줘.

### 3. 결과 요약 출력
각 레포별로 아래 형식으로 결과를 출력해줘:

```
✅ WorkSpace     — {브랜치명} @ {커밋 해시 7자리} ({커밋 메시지})
✅ LLM_wiki      — {브랜치명} @ {커밋 해시 7자리} ({커밋 메시지})
✅ AlgoNotion_FE — {브랜치명} @ {커밋 해시 7자리} ({커밋 메시지})
✅ AlgoNotion_BE — {브랜치명} @ {커밋 해시 7자리} ({커밋 메시지})
✅ TalkTime      — {브랜치명} @ {커밋 해시 7자리} ({커밋 메시지})
```

실패한 레포는 ✅ 대신 ❌ 로 표시하고 오류 원인을 함께 출력해줘.
