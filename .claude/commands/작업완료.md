# 작업 완료

$ARGUMENTS 에 task_id가 전달됩니다 (예: `/작업완료 AlgoNotion-Store`).

## 절차

### 1. 인자 및 기본 정보 확인
task_id = "$ARGUMENTS"
task_id가 비어있으면 "task_id를 입력해주세요. 예) /작업완료 AlgoNotion-Store" 안내 후 중단.

`config.local.json`에서 member를 읽어줘.
`projects/` 폴더의 모든 .json 파일에서 해당 task_id를 가진 태스크 정보(title, tags 등)와 project id를 찾아줘.

### 2. 세션 로그 분석
현재 디렉토리의 `.session-log.jsonl`을 읽어줘.
파일이 없으면 "세션 로그가 없습니다. /작업시작을 먼저 실행했는지 확인해주세요." 안내 후 중단.

다음 항목을 분석해줘:
- 시작 시각 (session_start의 ts)
- 종료 시각 (현재 시각)
- 총 소요 시간 (분)
- 에러 발생 횟수 (result가 "error"인 항목 수)
- 에러 내용 목록
- 수정된 파일 목록
- 총 사용 토큰 (알 수 없으면 0)

### 3. 로그 기록
WorkSpace 루트의 `logs/log-{member}.jsonl`에 아래 형식으로 한 줄 append:
```json
{"ts":"{ISO8601 현재시각}","member":"{member}","project":"{project}","task_id":"{task_id}","task":"{task title}","type":"feature","tokens":{토큰수},"duration_min":{소요분},"tags":[],"error_count":{에러수},"errors":[{에러내용 목록}],"files_touched":[{수정파일 목록}],"notes":""}
```

### 4. events.jsonl에 완료 이벤트 기록
```
{"ts":"{ISO8601 현재시각}","type":"task_done","member":"{member}","task_id":"{task_id}"}
```

### 5. 세션 로그 삭제
`.session-log.jsonl` 파일을 삭제해줘.

### 6. 완료 요약 출력
```
✅ {task_id} 완료!
- 소요 시간: {duration_min}분
- 수정 파일: {files_touched 수}개
- 에러 발생: {error_count}건
{에러가 있었다면 에러 목록도 출력}
```

### 7. 오늘 하루 마감
오늘 날짜(YYYY-MM-DD)를 구해줘.

**완료 태스크 요약:**
`events.jsonl`에서 오늘 날짜 기준 해당 member의 `task_done` 이벤트를 모두 찾아줘.
`logs/log-{member}.jsonl`에서 오늘 날짜 기준 기록을 읽어서 출력:

```
📊 오늘 {member}의 작업 요약 — {날짜}

✅ 완료한 태스크:
  - {task_id}: {task title} ({duration_min}분)
  ...

📈 통계:
  - 총 작업 시간: {합계}분
  - 총 에러 수: {합계}건
  - 수정 파일 수: {합계}개
```

**미완료 태스크 이월:**
`tasks/today-{날짜}-{member}.md`를 읽어서 오늘 완료 목록에 없는 태스크마다 events.jsonl에 append:
```
{"ts":"{ISO8601 현재시각}","type":"task_carry_over","member":"{member}","task_id":"{task_id}"}
```

**git 커밋 및 push:**
다음 파일만 스테이징해서 커밋 후 push:
- `logs/log-{member}.jsonl`
- `events.jsonl`

커밋 메시지: `log: {member} {날짜} (완료 {N}건, 이월 {M}건)`

```bash
git add logs/log-{member}.jsonl events.jsonl
git commit -m "log: {member} {날짜} (완료 {N}건, 이월 {M}건)"
git push
```

위 명령어를 실제로 실행해줘.

### 8. 마무리
"수고했어요! 내일도 파이팅! 👋" 출력.
